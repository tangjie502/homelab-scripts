# homelab-scripts

用于存放个人家庭实验室、Linux 服务器管理、网络排障、代理配置以及日常自动化运维脚本的仓库。

## 仓库说明

这个仓库主要用于统一管理平时常用的脚本，方便做版本控制、持续维护和后续扩展。

当前主要面向以下场景：

- Ubuntu / Linux 系统管理
- 网络与代理配置
- 服务维护与排障
- 家庭实验室日常运维
- 常用 Shell 工具脚本沉淀

这个仓库的目标，不只是保存零散脚本，而是逐步沉淀成一个可长期维护、可复用的个人脚本工具库。

---

## 当前已收录脚本

### 1. `proxy_manager.sh`

一个用于 Ubuntu 系统的交互式代理管理脚本。

#### 功能特性

- 查看当前系统代理状态
- 查看当前 APT 代理状态
- 设置系统代理
- 设置 APT 代理
- 取消系统代理
- 取消 APT 代理
- 通过菜单交互操作
- 设置代理时手动输入代理地址和端口
- 在写入配置前自动验证代理连通性和可用性

#### 适用场景

- Ubuntu 22.04 服务器
- Linux 虚拟机代理配置
- 家庭局域网中已有 HTTP 代理服务器的环境
- 需要分别管理 shell 代理和 apt 代理的场景
